## 🚀 Tentang Saya

Halo! Saya Tri Wahyu Handoyo, mahasiswa S1 Pendidikan Teknik Mekatronika di Universitas Negeri Yogyakarta. Saya memiliki ketertarikan mendalam pada:
- 🤖 **Robotika & Sistem Tertanam (*Embedded Systems*)** - Mikrokontroler, IoT, dan desain 3D.
- 🎥 **Multimedia** - *Video editing* dan *content creation*.
- 🏆 Aktif sebagai anggota **Abhinaya UNY Robotics Team**, berfokus pada pengembangan robot untuk kompetisi nasional (KRTMI).

## ✨ Fitur Website

- **Desain Modern & Responsif:** Tampilan yang otomatis menyesuaikan ukuran layar (Desktop, Tablet, maupun Mobile).
- **Sidebar Profil Statis:** Navigasi profil dan tautan sosial media yang selalu terlihat saat di-scroll.
- **Timeline Interaktif:** Menampilkan alur riwayat pendidikan dan pencapaian kompetisi.
- **Integrasi YouTube (Embed):** Memutar langsung *playlist* portofolio video editing dan dokumentasi proyek (*YouTube Shorts*) tanpa harus keluar dari website.
- **Animasi Ringan:** Efek *fade-in* yang membuat website terasa lebih hidup dan dinamis saat pertama kali dimuat.

## 🛠️ Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi web dasar (*Vanilla*):
- **HTML5:** Untuk struktur dan semantik halaman.
- **CSS3:** Untuk tata letak (Flexbox & Grid), styling, dan animasi.
- **Google Fonts:** Menggunakan *font* `Poppins` untuk tipografi yang modern.
- **FontAwesome (v6.4.0):** Untuk ikon-ikon antarmuka dan sosial media.
